# Ad & Pamphlet Creator

A modern web app for generating video ads and designing pamphlets with AI assistance and Canva-like templates.

## Features
- Sidebar navigation for easy access
- Video Ad Generator with template selection and export
- Pamphlet Editor with live preview
- AI Assistant (ChatGPT-like)
- Responsive design for mobile and desktop

## How to Publish on GitHub Pages
1. Create a new GitHub repository (e.g., `ad-pamphlet-creator`).
2. Upload all files from your `webapp` folder (`index.html`, `styles.css`, `app.js`, etc.).
3. Commit and push to the `main` branch.
4. Go to **Settings > Pages** in your repo.
5. Set Source to `main` branch, `/ (root)` folder.
6. Save. Your app will be live at: `https://<your-username>.github.io/<repo-name>/`
